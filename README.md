# FinPort

A Stock Trading platform that allows users to track investments, manage holdings, view positions, place orders, and monitor market data in real-time.

## Features

### User Authentication
- Secure user registration and login
- JWT-based authentication with cookie support
- Password hashing using bcrypt

### Dashboard
- Overview of portfolio performance
- Real-time market data visualization
- Summary of holdings, positions, and orders

### Holdings Management
- View current stock holdings
- Track average price, current price, and profit/loss
- Detailed breakdown of each holding

### Positions Tracking
- Monitor open positions
- Real-time P&L calculations
- Position sizing and risk management

### Order Management
- Place buy and sell orders
- Order history and status tracking
- Real-time order execution

### Analytics & Charts
- Interactive charts using Chart.js
- Doughnut charts for portfolio allocation
- Vertical graphs for performance tracking

### Responsive Design
- Mobile-friendly interface
- Material-UI components for consistent UI
- Cross-browser compatibility

## Technology Stack

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - ODM for MongoDB
- **JWT** - Authentication
- **bcrypt** - Password hashing
- **Passport.js** - Authentication middleware

### Frontend (Landing Page)
- **React** - UI library
- **React Router** - Client-side routing
- **Axios** - HTTP client
- **React Toastify** - Notifications

### Dashboard
- **React** - UI library
- **Material-UI** - Component library
- **Chart.js** - Data visualization
- **React Router** - Navigation
- **Axios** - API communication
